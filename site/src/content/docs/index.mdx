## Enterprise capabilities

Floci Studio closes the gap with hosted *Enterprise* cloud emulators — all local,
all driveable from the cockpit, the REST API and the MCP server:

- **[Observability & Debugging](/enterprise/observability/)** — Dead Letter Queue
  management with one-click redrive, a Flight Recorder for time-travel debugging of
  in-flight events, and an X-Ray-style service graph of real event flows.
- **[IaC Drift Detection](/enterprise/iac-drift/)** — auto-discover Terraform /
  Serverless / CDK resources and detect drift between your code and the emulator.
- **[Hybrid Development](/enterprise/hybrid/)** — seed anonymized data from real
  AWS, proxy live SQS traffic to local functions, and open reverse tunnels for
  third-party webhooks.
- **[Extensibility & Plugins](/enterprise/extensibility/)** — lifecycle webhooks,
  declarative HTTP interceptors and a community plugin SDK.
